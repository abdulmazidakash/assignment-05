Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a webpage as a tree of objects, allowing developers to manipulate HTML and CSS. Through the DOM, scripts can dynamically change content, structure, and styles of a webpage.

How do you select an element from the DOM?

can select a DOM element using methods like:

  • getElementById('id')
  • getElementsByClassName('class')
  • getElementsByTagName('tag')
  • querySelector('selector') (for the first match), and
  • querySelectorAll('selector') (for all matches based on a CSS selector).
What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique where a single parent element handles events for its child elements using event bubbling. It improves performance by reducing the number of event listeners, especially for dynamically added elements, making it more efficient for handling large sets of elements.

How do you manipulate an element's attributes and styles using the DOM?

can manipulate an element's attributes using

  • setAttribute()
  • getAttribute()
  • removeAttribute()

    • Use the `style` property (`element.style.color = 'red'`) or manage classes with `classList` methods like `add()`, `remove()`, and `toggle()` to dynamically change styles.
